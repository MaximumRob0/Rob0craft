Rob0craft
Кубическая песочница от первого лица на Python с использованием Ursina и Pygame.

🎮 Возможности
Плоский мир 25x25 блоков

5 типов базовых блоков (трава, камень, дерево, листва, земля)

5 видов руд (алмаз, железо, золото, изумруд, уголь)

Враги-криперы (преследуют, шипят, взрываются, отбрасывают игрока)

Система текстурпаков

Главное меню на Pygame

📦 Установка
bash
# Клонирование репозитория
git clone https://github.com/yourusername/rob0craft.git
cd rob0craft

# Установка зависимостей
pip install ursina pygame pillow

# Запуск игры
python rob0craft.py
🎮 Управление
Клавиша	Действие
WASD	Перемещение
Мышь	Поворот камеры
ЛКМ	Сломать блок
ПКМ	Поставить блок
1-5	Выбор блока (трава/камень/дерево/листва/земля)
6-0	Выбор руд
R	Телепортация на старт
ESC	Выход
🎨 Кастомизация
Текстурпаки
Создайте папку texturepacks в корне игры (если её нет)

Скопируйте папку default и переименуйте

Замените PNG-файлы на свои текстуры (размер 16x16 или 32x32)

Выберите текстурпак в главном меню

Доступные текстуры:

grass.png / stone.png / oak.png / leaves.png / dirt.png

bedrock.png / creeper.png

diamond_ore.png / iron_ore.png / gold_ore.png / emerald_ore.png / coal_ore.png

Небо
Поместите файл sky.jpg в папку вашего текстурпака для кастомного неба.

Как добавить новый блок
Создайте класс в разделе "КЛАССЫ БЛОКОВ":

python
class NewBlock(Entity):
    def __init__(self, pos):
        tex = textures_cache.get("new_block")
        if tex:
            super().__init__(model='cube', texture=tex, position=pos, collider='box')
        else:
            super().__init__(model='cube', color=color.rgb(100,100,100), position=pos, collider='box')

Как изменить размер мира
Найдите в коде:

python
for x in range(-12, 13):
    for z in range(-12, 13):
Измените числа 12 на желаемый радиус. Мир станет (радиус*2+1) x (радиус*2+1) блоков.

Как изменить скорость/поведение крипера
В классе Creeper:

self.speed = 0.06 — скорость движения

self.fuse_time = 35 — время до взрыва (чем меньше, тем быстрее)

Дистанции в update(): dist < 1.5 (подход), dist < 5 (шипение)
